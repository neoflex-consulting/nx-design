## Текст

### Цвет

`view` отвечает за тип и цвет текста.

```ts
<Typography view="alert">Газпром нефть</Typography> /* ошибка */
<Typography view="brand">Газпром нефть</Typography> /* цвет бренда */
<Typography view="ghost">Газпром нефть</Typography> /* примечания */
<Typography view="link">Газпром нефть</Typography> /* ссылка */
<Typography view="link-minor">Газпром нефть</Typography> /* дополнительная ссылка */
<Typography view="primary">Газпром нефть</Typography> /* основной текст */
<Typography view="secondary">Газпром нефть</Typography> /* дополнительный */
<Typography view="success">Газпром нефть</Typography> /* успешное действие */
<Typography view="warning">Газпром нефть</Typography> /* предупреждение */
```

### Размер

`size` отвечает за размер текста.

- По умолчанию в параграфах используется размер `m`.
- Даже для самых неважных подсказок используйте размер `xs` или больше.
- `2xs` — только для текста в [верхнем регистре](#регистр).

```ts
<Typography size="2xs">Нефть</Typography>
<Typography size="xs">Нефть</Typography>
<Typography size="s">Нефть</Typography>
<Typography size="m">Нефть</Typography>
<Typography size="l">Нефть</Typography>
<Typography size="xl">Нефть</Typography>
<Typography size="2xl">Нефть</Typography>
<Typography size="3xl">Нефть</Typography>
<Typography size="4xl">Нефть</Typography>
<Typography size="5xl">Нефть</Typography>
<Typography size="6xl">Нефть</Typography>
```

<TextExampleSize />

### Гарнитура

`font` отвечает за гарнитуру шрифта.

Вы можете выбрать гарнитуру, заданную в системе:

- `primary` меняет шрифт на основной шрифт дизайн-системы,
- `mono` — на шрифт, заданный как моноширный.

Это нужно, чтобы расставить дополнительные акценты: например, выделить что-то важное
прямо в тексте.

### Насыщенность

`weight` отвечает за насыщенность (жирный, полужирный, обычный, светлый и другие варианты).

> Убедитесь, что в шрифте есть насыщенность, которую вы используете. Браузер имитирует
> насыщенность, которой нет в наборе, но неизвестно, как это будет выглядеть.

```ts
<Typography weight="black">Газпром нефть</Typography>
<Typography weight="bold">Газпром нефть</Typography>
<Typography weight="semibold">Газпром нефть</Typography>
<Typography weight="regular">Газпром нефть</Typography>
<Typography weight="light">Газпром нефть</Typography>
<Typography weight="thin">Газпром нефть</Typography>
```

### Межбуквенный интервал

`spacing` меняет межбуквенный интервал.

```ts
<Typography spacing="xs">
  «Газпром нефть» ведет работу в крупнейших российских нефтегазовых регионах.
</Typography>
<Typography spacing="s">
  «Газпром нефть» ведет работу в крупнейших российских нефтегазовых регионах.
</Typography>
<Typography spacing="m">
  «Газпром нефть» ведет работу в крупнейших российских нефтегазовых регионах.
</Typography>
<Typography spacing="l">
  «Газпром нефть» ведет работу в крупнейших российских нефтегазовых регионах.
</Typography>
```

## Контейнер с текстом

### Тег для контейнера

По умолчанию `Typography` представлен в HTML-документе в виде тега `div`. Вы можете его
переопределить с помощью свойства `as`.

### Расположение

`display` определяет, как будет вести себя контейнер с текстом по отношению к другим элементам
на странице (как строчный или блочный элемент). Варианты: `block`, `inline-block`, `inline`.

## Блоки с текстом

### Абзац и заголовок

Пример классической конструкции: абзац с заголовком.

```ts
<Typography
  as="div"
  align="left"
  font="sans"
  lineHeight="l"
  size="2xl"
  spacing="m"
  transform="uppercase"
  view="primary"
  weight="bold"
>
  Заголовок несет суть
</Typography>
<Typography as="p" align="left" font="serif" lineHeight="s" size="s" view="primary">
  Абзац — это часть текста между отступами с новой строки. Абзац можно назвать микротекстом.
  В общей теме могут выделяться и микротемы. При этом каждый абзац раскрывает свою
  микротему. Что же ещё можно сказать про абзацы? Мы видим, что обычно в абзацах несколько
  предложений. В таком случае каждый абзац служит для выделения своей микротемы.
</Typography>
```

### Форматирование текста в абзаце

Вы можете выделять отдельные слова, фразы или части фраз с помощью форматирования.
Например, чтобы обратить внимание на важную информацию, выделить термин, элемент
интерфейса, прямую речь.

```ts
<Typography as="div" align="left" font="sans" lineHeight="l" size="m" view="primary">
  <Typography as="span" fontStyle="italic">
    Примечание.&nbsp;
  </Typography>
  База скважин сформирована на основе отчетов сотрудников. Любые&nbsp;
  <Typography weight="black" as="span">
    срочные изменения
  </Typography>{' '}
  должны вноситься по регламенту в сроки, указанные Центром Разработок.
</Typography>
```

## Пример использования

Использование компонента:

```ts
// src/App.ts
import React from 'react';
import { Typography } from '@consta/uikit/Typography';

export function App() {
  return (
    <Typography
      as = 'p'
      align = 'center'
      cursor = 'default'
      display = 'block'
      font = 'mono'
      lineHeight = '2xs'
      size = 'm'
      spacing = 'xs'
      view = 'primary'
      weight = 'regular'
      width = 'default'
      className = 'custom-class-name'
    >
      Текст
    </>
  );
}
```
